# Google Hashcode 2021

## Startup

Input folder: `\input`.

Solutions will be written to `\output` (created by UnitTest).


```
.
├── input
    ├── f1.txt
    ├── f2.txt
    ├── f3.txt
└── output
    ├── f1.txt
    ├── f2.txt
    ├── f3.txt
```
